.. ifconfig:: not autoapi_available

   .. note::

      Auto-generated API documentation is disabled because ``sphinx-autoapi`` is not installed in this environment.
      Install the docs extras before building locally::

         pip install -e .[docs]

      Once ``sphinx-autoapi`` is available, rebuild the docs (``make html`` or ``sphinx-autobuild``) to generate the reference.
